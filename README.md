# Portafolio Personal

Este proyecto es un portafolio web interactivo y moderno diseñado para mostrar mi experiencia, proyectos y habilidades técnicas. Esta construido utilizando React y Vite, enfocandose en el rendimiento y una experiencia de usuario fluida.

## Caracteristicas Principales

- **Diseño Responsivo**: Adaptable a diferentes tamaños de pantalla y dispositivos.
- **Internacionalizacion**: Soporte completo para multiples idiomas (Español e Ingles), gestionado a traves de un contexto de React.
- **Tema Claro y Oscuro**: Funcionalidad para cambiar entre modos de visualizacion claro y oscuro.
- **Navegacion Suave**: Desplazamiento fluido entre las diferentes secciones de la pagina.
- **Animaciones**: Transiciones y efectos visuales integrados con Framer Motion.
- **Formulario de Contacto**: Integracion funcional con Formspree para el envio de mensajes.

## Tecnologias Utilizadas

- **React 19**: Biblioteca principal para la construccion de la interfaz de usuario.
- **Vite**: Entorno de desarrollo rapido y herramienta de construccion.
- **Framer Motion**: Biblioteca para animaciones de produccion en React.
- **React Icons**: Coleccion de iconos populares para proyectos React.
- **React Scroll**: Utilidad para animar el desplazamiento vertical.

## Estructura del Proyecto

El codigo fuente se organiza de la siguiente manera:
- **components**: Componentes reutilizables como la barra de navegacion (Navbar) y el pie de pagina (Footer).
- **sections**: Componentes principales que conforman las secciones de la pagina (Inicio, Sobre Mi, Proyectos, Habilidades, Certificaciones, Contacto).
- **context**: Contextos de React para manejar el estado global del idioma y el tema.
- **data**: Archivos JSON y de datos estaticos para el contenido.

# Personal Portfolio

This project is an interactive and modern web portfolio designed to showcase my experience, projects, and technical skills. It is built using React and Vite, focusing on performance and a smooth user experience.

## Key Features

- **Responsive Design**: Adaptable to different screen sizes and devices.
- **Internationalization**: Full support for multiple languages (Spanish and English), managed through a React context.
- **Light and Dark Theme**: Functionality to toggle between light and dark viewing modes.
- **Smooth Navigation**: Smooth scrolling between different page sections.
- **Animations**: Transitions and visual effects integrated with Framer Motion.
- **Contact Form**: Functional integration with Formspree for message sending.

## Technologies Used

- **React 19**: Main library for building the user interface.
- **Vite**: Fast development environment and build tool.
- **Framer Motion**: Library for production-ready animations in React.
- **React Icons**: Collection of popular icons for React projects.
- **React Scroll**: Utility to animate vertical scrolling.

## Project Structure

The source code is organized as follows:
- **components**: Reusable components such as the Navbar and Footer.
- **sections**: Main components that make up the page sections (Hero, About, Projects, Skills, Certifications, Contact).
- **context**: React contexts to handle global state for language and theme.
- **data**: JSON and static data files for content.
